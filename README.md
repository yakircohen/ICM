# Usage
1. Download 
2. Load icm.sql to mysql server
3. Open CMD and run the following command 
```
java -jar G11_server.jar
```
4. Tick "Run Server"
5. Open CMD and run the following command 
```
java -jar G11_client.jar
```
6. Login
